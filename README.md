# 教务管理系统
> 软件开发方法实验课程设计

## 编译环境
* Windows 10
* Visual Studio 2019  
* .Net Framework 4.7.2

## UI框架
* 使用WPF和MVVM框架

## 编译步骤
1. 从[MySQL官网](https://www.mysql.com/)下载[MySql.Data.dll](https://dev.mysql.com/downloads/windows/visualstudio/)，并添加到项目引用
2. 在Nuget中添加Prism框架
